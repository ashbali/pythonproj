How to run the code:
Run stock_predictor_cli.py in the terminal.

Description of the python files:
stock_predictor_cli.py is the main python file that is used for running the application on CLI. 

descriptive.py contains functions that shows descriptive analysis of the stocks.

predictive.py contains functions to predict closing price the stocks. 

companylist.csv contains all the NASDAQ listed company information in csv format.

terms.txt contains terms & conditions of the program.
